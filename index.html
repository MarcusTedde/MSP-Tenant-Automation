Before You Begin

  
This tool connects to your Microsoft 365 tenant via the Graph API. To make that work, you need three things — and this wizard will walk you through each one.





  

    

      
1

      
Create an App Registration

      
A one-time 60-second task in the Azure Portal. This gives the tool an identity in your tenant.

    

  

  

    

      
2

      
Paste the Client ID & Sign In

      
Copy the Application ID from Azure, paste it here, and sign in as Global Admin.

    

  

  

    

      
3

      
Run the Permission Bootstrap

      
One click — the tool automatically configures all required Graph API permissions and generates a client secret.

    

  

  

    💡 You'll need: Global Administrator access to the target Microsoft 365 tenant.
  

  

    
    
  











  

    
1

    STEP 1 OF 3

      Automatic Setup (Recommended)

Connect to Your Tenant
